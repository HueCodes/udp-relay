# udp-relay

UDP relay is a high-throughput MAVLink v2 telemetry aggregator written in Go. It listens for UDP packets on port 14550, parses and validates MAVLink frames with CRC-16/MCRF4XX checksums, applies per-drone token bucket rate limiting and optional source IP/CIDR whitelisting, then maintains a thread-safe drone registry tracking position, attitude, battery, arm state, and flight mode for up to 250 vehicles. A worker pool fans parsed events through a non-blocking pub/sub hub to WebSocket clients that can subscribe to filtered drone subsets, while a REST API exposes drone state and per-component health. Prometheus metrics, optional pprof endpoints, YAML configuration with CLI overrides, and a distroless Docker image with Compose-based Prometheus/Grafana monitoring round out the deployment story, all running as a single zero-dependency binary (plus nhooyr.io/websocket and prometheus/client_golang).
